# Star Blood Attribute Diagram

Интерактивная концентрическая диаграмма атрибутов на `D3.js`.

## Что внутри

- `index.html` — визуализация диаграммы и вся логика рендера/интерактива.
- `attributes.data.json` — локальный источник данных атрибутов.

## Возможности

- 5 рангов слоев: `дерево -> бронза -> серебро -> золото -> небо`.
- Центр: `Источник` + 3 базовых атрибута (`Тело`, `Разум`, `Дух`).
- Фокус по клику:
  - подсветка родителя/выбранного/детей,
  - поворот модели (родитель снизу, дети сверху),
  - авто-приближение области фокуса.
- Кнопка `Сброс` возвращает исходный вид.
- Hover-tooltip: `Ранг`, `Атрибут`, `Описание`, `Скиллы` (если есть).
- Космический анимированный фон, синхронизированный с движением камеры.

## Запуск

Рекомендуется запускать через локальный HTTP-сервер (из-за загрузки JSON через `fetch`):

```bash
python -m http.server 8000
```

После этого открыть:

- [http://localhost:8000/index.html](http://localhost:8000/index.html)

## Формат данных (`attributes.data.json`)

Узел атрибута:

```json
{
  "title": "Название",
  "description": "Описание",
  "skills": "Навыки",
  "children": []
}
```

Правила:

- Если `title` или `description` отсутствуют, используется `неизвестно`.
- Для промежуточных рангов обычно 3 дочерних узла.
- `небо` — конечный ранг (листовой узел).
